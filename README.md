# LendSwift Multi-Step Loan Application Form

## Features

- 9-Step Loan Application Wizard
- React Hook Form + Zod Validation
- Personal, Home, Business Loan Types
- PAN & Aadhaar Verification Simulation
- PIN Code Address Auto-Fill
- Conditional Form Rendering
- Document Upload & Preview
- E-Signature Capture
- Auto Save & Resume
- EMI Calculator
- Pre-Approval Score
- Application Summary & Print
- Responsive Design

## Tech Stack

- React
- TypeScript
- Vite
- Zustand
- React Hook Form
- Zod
- Tailwind CSS
- React Dropzone
- React Signature Canvas
